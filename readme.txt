A simple plugin to work with onePlace Tag Module.

== Description ==

If you have a onePlace with active Tag Module, this Plugin is for you!

Use your onePlace Tags in Wordpress and Elementor

* Is working with any onePlace Tag

It comes with some basic shortcodes and elementor widgets

* Tag List Widget

OnePlace Connect Plugin is required !

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enable the settings you want in "WP PLC Tag" in the OnePlace Menu in Settings Panel

== Frequently Asked Questions ==

= What can I do with this plugin? =

Use your onePlace Tags like Categories in Wordpress.

= How do I setup this plugin? =

Just install and activate. no setup or configuration yet.
